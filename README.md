## Maxxidata Airflow - Tutorial

#### First class - 29-Abr-2022

Before start:

- Download the database called Chinook - [download](https://github.com/lerocha/chinook-database)
- Create a SQLite file called `chinook_db.sqlite`
- Run `chinook.sql` on `chinook_db.sqlite`
- Then, configure on Airflow the connection called `chinook_db`

#### Dependencies:

- Docker Compose - [instructions](https://docs.docker.com/compose/install/)

#### Commands:

```
$> cd path-to-project
$> docker-compose build
$> docker-compose up
```
